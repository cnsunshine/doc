# 1. 错误码
## 1.1. 用户权限
| 状态码 | 状态信息 | 描述 |
| --- | --- | --- |
| 20001 | 请重新获取token | 刷新token |
| 20002 | 参数不完整 | &nbsp; |
| 20003 | 注册参数不完整或不符合标准 | 用户名5-10位字符串，密码5-10位字符串，也可能用户名重复 |
| 20004 | 注册时增加用户失败 | 可能是有重复 |
| 20005 | 登录密码错误 | &nbsp; |
| 20006 | redis故障 | 检查redis故障 |
| 20007 | 数据库修改失败 | 检查数据库故障 |
| 20008 | 数据格式不正确 | 检查数据格式是否正确 |
