# 1. 约定 
## 1.1. 交互 
前后端完全数据分离，采用**json**进行数据交换
## 1.2. 用户认证
| key | description |
| --- | --- | 
| x-token | 登录token令牌 | 
| x-csrf-token | 页面提交防止csrf攻击token | 
## 1.3. 时间
交互使用时间戳，精确到**秒** 
## 1.4. 数据请求格式 
| 参数 | 类型 | 描述 | 
| --- | --- | --- | 
| {参数1} | {类型1} | 参数1 
| {参数2} | {类型2} | 参数2 
示例：
```json
{
    "username":123,
    "password":"pwd"
} 
```
## 1.5. 数据返回格式 
| 参数 | 类型 | 描述 |
| --- | --- | --- |
| code | int | 返回状态码 | 
| body | json | 数据内容，根据实际接口定义 | 
示例：
```json
{
    "code":200,
    "body":{
        "uid":"123456",
    }
}
```

| 返回值 | 状态码 | 描述 | 备注 |
